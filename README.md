# official webSite 

SPT (Sport Token) is a decentralized sports prediction protocol built on Binance Smart Chain (BSC). By reconstructing traditional sports prediction markets with Pari-mutuel (pool betting) logic, we eliminate the head-to-head risk of centralized bookmakers and create a truly transparent, fair, and value-sharing Web3 sports ecosystem.

On the SPT platform, every token holder is not only a participant but also a “shareholder” of the platform.

🌟 Core Features | Core Features
No House Risk: Uses an equal pool (pari-mutuel) system. The platform does not take positions; it only serves as a neutral matching protocol.

Real Yield: A 3% fee charged on each transaction is distributed directly to SPT stakers, enabling “hold to earn dividends.”

Ready for the 2026 World Cup: Fully compatible with the new 48-team World Cup format, supporting multiple play types such as champion prediction, single-match 1X2 (win/draw/lose), total goals, and more.

Maximum Transparency: All odds calculation, fund flows, and settlement logic are executed on-chain and publicly verifiable.
